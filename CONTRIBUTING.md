# 🤝 Contributing to Advora App

Thank you for your interest in contributing to Advora!  
This document outlines the workflow, coding standards, and contribution guidelines followed by our team.

---

## 📌 Contribution Workflow

1. Fork the repository (if external contributor)
2. Create a new branch:
git checkout -b feature/your-feature-name
3. Make your changes
4. Commit with a meaningful message
5. Push your branch:
git push origin feature/your-feature-name
6. Create a Pull Request (PR) for review

---

## 🌿 Branching Strategy

| Branch | Purpose |
|--------|--------|
| main | Stable production-ready code |
| Mahek-updates | New Features |
| mishti(adminside) | Admin Panel Code |
## 📝 Commit Message Guidelines

Use **clear and meaningful commit messages** in present tense:

### ✅ Good Examples:
- Add login screen validation  
- Fix Firebase authentication bug  
- Implement search filter feature  

### ❌ Avoid:
- fix  
- update  
- changes  
## 🧩 Code Guidelines

- Follow consistent naming conventions  
- Write clean and readable code  
- Avoid unused variables and commented code  
- Maintain proper indentation and formatting  

---

## 🔍 Pull Request Guidelines

Before creating a PR:
- Ensure the app builds and runs correctly  
- Test your feature/bug fix  
- Provide a clear description of changes  

PR should include:
- What was implemented/fixed  
- Screenshots (if UI changes)  

---

## 🧪 Testing

- Test all features before pushing  
- Ensure no existing functionality breaks  
- Check UI consistency across screens  

---

## 🔐 Security

- Do NOT commit API keys, credentials, or sensitive data  
- Use environment variables for configuration  

---

## 👩‍💻 Team Collaboration

- All major changes should be discussed before implementation  
- Follow the agreed design system and architecture  
- Maintain clear communication among team members  

---

## 🚀 Future Contributions

We welcome improvements such as:
- UI/UX enhancements  
- Performance optimization  
- Feature additions  

---

Thank you for contributing to **Advora App** 💡
